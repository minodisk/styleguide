# Overview

### 命名

- 一般的でない省略をしない。数文字の入力のコストよりも、他人が省略された文字列から元の単語を推測するコストが圧倒的に大きいため。
- カラフルな単語を選択する。

### TODO コメント

TODO でタスクの明確化を図る。

```html
<!-- TODO(JSer): スライダの実装 -->
```
```css
/* TODO(JSer): 動的に高さを設定 */
```
```javascript
// TODO(JSer): ロード後に呼び出す
```


## ファイル構成

### エンコード

UTF-8(BOM無し)

### 改行コード

LF

### git

- [.gitignore](https://github.com/trymore/gitignore) をリポジトリのルートに置く


## エディタ

- 空白文字(タブ, 半角スペース, 全角スペース)を表示できるエディタを使う。不要な空白文字でエラーを出さないため。
